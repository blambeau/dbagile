h1. DbAgile - Making our Databases Agile

DbAgile aims at providing agile tools around databases, and relational/sql databases in particular. It can be used as:

* a lightweight yet agile relational database server
* an agile ruby architecture around SQL databases
* a friendly command line tool for connecting/manipulating SQL databases

h2. Main features

* agile database schema: allow schema evolutions without issuing alter/create table commands
* agile database types: allow user-defined types to be included in all tables

h2. The command line tool

<pre class="shell">
  bl@shell# dbagile --full-agile 

  Welcome in dbagile interactive terminal.

  Type:  \c to connect a database
         \h for help
         \q to quit

  dbagile=# \c sqlite://example.db
  dbagile=# plug Defaults, {:now => Defaults::now}
  dbagile=# insert :tools, {:name => "dbagile", :version => "0.0.1"}
  dbagile=# display :tools
  
  -------------------------------------------------------------------
  | name: String | version: String | now: Time                      |
  -------------------------------------------------------------------
  | dbagile      | 0.0.1           | Thu Jun 10 14:18:31 +0200 2010 |
  -------------------------------------------------------------------
</pre>
